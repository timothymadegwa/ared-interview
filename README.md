# ared-interview
 
### Tech/Framework
> Python 3.8.11, Django 3.2.7
### Getting started 
> git clone the repo  
> python3 -m venv ./venv  (Create venv)  
> pip install -r requirements.txt (Install python packages)  
> python manage.py makemigrations  
> python manage.py migrate  
> python manage.py runserver\
